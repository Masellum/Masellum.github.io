数论学习笔记
2018-09-07 09:39   folder 学习笔记 label 数学, 学习笔记, 数论

基本定义

对于两个自然数 a,ba, ba,b，如果 a=kba = kba=kb 且 b≠0b \neq 0b​=0，那么我们说 bbb 整除 aaa 或者说 aaa 被 bbb 整除，记作 b∣ab \mid ab∣a；否则 bbb 不整除 aaa，记作 b∤ab \nmid ab∤a。如果 b∣ab \mid ab∣a，那么我们说 bbb 是 aaa 的因数，aaa 是 bbb 的倍数。如果一个数同时是两个数的因数，那么这个数是这两个数的公因数。如果两个数除了 111 之外没有别的公因数，那么这两个数互质，记作 a ⊥ ba \ \bot \ ba ⊥ b。

欧几里得算法

欧几里得算法又称辗转相除法，用来求两个数的最大公因数。依据是 gcd⁡(a,b)=gcd⁡(b,a % b)\gcd(a, b) = \gcd(b, a\ \% \ b)gcd(a,b)=gcd(b,a % b)。

证明：

gcd⁡(a,b)=gcd⁡(b,a % b)  ⟺  gcd⁡(a,b)=gcd⁡(b,a−b)\gcd(a, b) = \gcd(b, a\ \% \ b) \iff \gcd(a, b) = \gcd(b, a - b)gcd(a,b)=gcd(b,a % b)⟺gcd(a,b)=gcd(b,a−b)。使用反证法证明。设 r=gcd⁡(a,b)r = \gcd(a, b)r=gcd(a,b)，则 rrr 也是 b,a−bb, a - bb,a−b 的因数。如果 r≠gcd⁡(b,a−b)r \neq \gcd(b, a - b)r​=gcd(b,a−b)，那么一定有一个数 sss 满足 s=gcd⁡(b,a−b)s = \gcd(b, a - b)s=gcd(b,a−b)。此时 s∣bs \mid bs∣b 且 s∣a−bs \mid a - bs∣a−b，故 s∣as \mid as∣a，故 sss 也是 a,ba, ba,b 的公因数。但 s>r=gcd⁡(a,b)s \gt r = \gcd(a, b)s>r=gcd(a,b)，产生矛盾，故不可能有符合条件的 sss，故 r=gcd⁡(b,a−b)r = \gcd(b, a - b)r=gcd(b,a−b)。

证毕。

代码：

int gcd(int a, int b) {
    return b == 0 ? a : gcd(b, a % b);
}

时间复杂度为对数级别。

关于最大公约数还有一个性质是 gcd⁡(a,b)×lcm⁡(a,b)=a×b\gcd(a, b) \times \operatorname{lcm}(a, b) = a \times bgcd(a,b)×lcm(a,b)=a×b。

证明：

根据算术基本定理，a,ba, ba,b 均可表示成如下形式：

a=p1a1p2a2…pnanb=p1b1p2b2…pmbm \begin{aligned} a = p_1^{a_1}p_2^{a_2} \ldots p_n^{a_n}\\ b = p_1^{b_1}p_2^{b_2} \ldots p_m^{b_m}\\ \end{aligned} a=p1a1​​p2a2​​…pnan​​b=p1b1​​p2b2​​…pmbm​​​

其中 pip_ipi​ 为质数。我们将 aaa 和 bbb “对齐”，也就是将他们的质因数逐个对应，只在一个数里出现过的令其指数为 000。显然

gcd⁡(a,b)=∏i=1max⁡(n,m)pimin⁡(ai,bi)lcm⁡(a,b)=∏i=1max⁡(n,m)pimax⁡(ai,bi) \begin{aligned} \gcd(a, b) = \prod_{i = 1}^{\max(n, m)}p_i^{\min(a_i, b_i)}\\ \operatorname{lcm}(a, b) = \prod_{i = 1}^{\max(n, m)}p_i^{\max(a_i, b_i)} \end{aligned} gcd(a,b)=i=1∏max(n,m)​pimin(ai​,bi​)​lcm(a,b)=i=1∏max(n,m)​pimax(ai​,bi​)​​

故显然 gcd⁡(a,b)×lcm⁡(a,b)=a×b\gcd(a, b) \times \operatorname{lcm}(a, b) = a \times bgcd(a,b)×lcm(a,b)=a×b。

扩展欧几里得算法

扩展欧几里得算法用来求解形如 ax+by=gcd⁡(a,b)ax + by = \gcd(a, b)ax+by=gcd(a,b) 的不定方程。 ax+by=gcd⁡(a,b)=gcd⁡(b,a % b) \begin{aligned} ax + by &= \gcd(a, b) \\[2ex] &= \gcd(b, a \ \% \ b) \\[2ex] \end{aligned} ax+by​=gcd(a,b)=gcd(b,a % b)​ 因此我们可以列出新的方程 bx′+(a % b)y′=gcd⁡(b,a % b)=bx′+(a−⌊ab⌋×b)y′=ay′+b⋅(x′−⌊ab⌋×y′) \begin{aligned} bx' + (a \ \% \ b)y' &= \gcd(b, a \ \% \ b) \\[2ex] &= bx' + (a - \lfloor \frac{a}{b} \rfloor \times b) y' \\[2ex] &= ay' + b \cdot (x' - \lfloor \frac{a}{b} \rfloor \times y') \end{aligned} bx′+(a % b)y′​=gcd(b,a % b)=bx′+(a−⌊ba​⌋×b)y′=ay′+b⋅(x′−⌊ba​⌋×y′)​

故 x=y′,y=x′−⌊ab⌋×y′x = y', y = x' - \lfloor \frac{a}{b} \rfloor \times y'x=y′,y=x′−⌊ba​⌋×y′。所以我们可以在进行欧几里得算法的同时递归得出每一组 a,ba, ba,b 所对应的 x,yx, yx,y 。递归终止条件是当 b=0b = 0b=0 时 x=1,y=0x = 1, y = 0x=1,y=0。其实 yyy应该是可以随便取的，不过大家都取 000（可能是为了方便计算）。这样便可得到不定方程的一组解。已经求出一组 x,yx, yx,y 以后，ax+by=a(x−b)+b(y+a)ax + by = a(x - b) + b(y + a)ax+by=a(x−b)+b(y+a)，所以我们可以由一组解计算出其他解。

代码：

int exgcd(int a, int b, int &x, int &y) {
    if (b == 0) {
        x = 1, y = 0;
        return a;
    }
    int res = exgcd(b, a % b, y, x);
    y -= x * a / b;
    return res;
}

扩展欧几里得算法还可以用来计算 ax≡1(modb)ax \equiv 1 \pmod bax≡1(modb) 的同余方程。因为这种方程可以化为 ax+by=1ax + by = 1ax+by=1，当 gcd⁡(a,b)=1\gcd(a, b) = 1gcd(a,b)=1 即 a,ba, ba,b 互质的时候此方程有解，其他情况无解。

质数

质数的定义不再重复。

算术基本定理

任意一个正整数都可以表示成如下形式： n=p1a1p2a2…pnan n = p_1^{a_1}p_2^{a_2} \ldots p_n^{a_n} n=p1a1​​p2a2​​…pnan​​ ，其中 ppp 是质数。

质因数分解

分解 nnn 的质因数的方法是，枚举所有小于等于 n\sqrt{n}n​ 的数，如果这个数是 nnn 的因数，那么就把这个数从 nnn 中除去并记录它的指数增加了一；如果除完后这个数仍能整除 nnn，那么继续除到无法整除为止。枚举完这些数以后如果 nnn 被除后剩余的商仍不等于一，那么这个剩余的数就是 nnn 的唯一一个大于 n\sqrt{n}n​ 的质因数。这个算法的正确性十分显然，不再证明。

根据乘法原理，如果 n=p1a1p2a2…pnann = p_1^{a_1}p_2^{a_2} \ldots p_n^{a_n}n=p1a1​​p2a2​​…pnan​​，那么 nnn 的因数个数为 ∏i=1n(ai+1)\prod_{i=1}^n (a_i+1)∏i=1n​(ai​+1)。

质数筛法

这里只记录线性筛相关知识，暴力筛和埃氏筛不再记录。

线性筛的基本思想是，每个合数只会被它的最小质因数筛去。

代码：

bool isPrime[MAXM];
int primes[7300], cnt, sum[MAXM];

void sieve(int m) {
    std::fill(isPrime + 1, isPrime + m + 1, true);
    isPrime[0] = isPrime[1] = false;
    for (int i = 2; i <= m; ++i) {
        if (isPrime[i]) {
            primes[++cnt] = i;
        }
        for (int j = 1; j <= cnt && i * primes[j] <= m; ++j) {
            isPrime[i * primes[j]] = false;
            if (i % primes[j] == 0) { // 线性之处
                break;
            }
        }
    }
}

线性筛还可以用来预处理出一系列数论函数的值。

逆元

因为除法在取模意义下不封闭，因此如果我们想在取模意义下做“除法”，就要用到逆元。

如果 a⋅a−1≡1(modp)a \cdot a^{-1} \equiv 1 \pmod pa⋅a−1≡1(modp)，那么我们就将 a−1a^{-1}a−1 记作 aaa 在 模 ppp 意义下的逆元。

求逆元的方法有若干种。

费马小定理

如果 ppp 是质数，那么 ap−1≡1(modp)a^{p - 1} \equiv 1 \pmod pap−1≡1(modp)。所以 a⋅ap−2≡1(modp)a \cdot a^{p-2} \equiv 1 \pmod pa⋅ap−2≡1(modp)，即 aaa 在模 ppp 意义下的逆元是 ap−2a^{p - 2}ap−2。可以用快速幂在 O(log⁡n)O(\log n)O(logn) 的时间复杂度内求出 aaa 的逆元。

线性求逆元

考虑取模的定义式： p mod a=p−⌊pa⌋×a p \bmod a = p - \lfloor \frac{p}{a} \rfloor \times a pmoda=p−⌊ap​⌋×a 将这个式子放在模 ppp 同余系下： p mod a≡−⌊pa⌋×a(modp) p \bmod a \equiv -\lfloor \frac{p}{a} \rfloor \times a \pmod p pmoda≡−⌊ap​⌋×a(modp) 两边同时乘以 (p mod a)−1⋅a−1(p \bmod a)^{-1} \cdot a^{-1}(pmoda)−1⋅a−1： a−1≡−⌊pa⌋×(p mod a)−1(modp) a^{-1} \equiv -\lfloor \frac{p}{a} \rfloor \times (p \bmod a)^{-1} \pmod p a−1≡−⌊ap​⌋×(pmoda)−1(modp) 因为 (p mod a)−1<a(p \bmod a)^{-1} \lt a(pmoda)−1<a，因此逆元可以线性递推得出。初始条件为 1−1=11^{-1} = 11−1=1。写代码的时候需注意负数取模。

int inv[MAXN];

void get_inv(int n, int p) {
    inv[1] = 1;
    for (int i = 2; i <= n; ++i) {
        inv[i] = 1LL * (p - p / i) * inv[p % i] % p;
    }
}

    #include <algorithm>
#include <cstdio>
#include <cstring>
#include <numeric>

const int MAXN = 50000 + 5;

int N, a, b, c, d, k;
bool isNotPrime[MAXN];
int primes[6000], mu[MAXN], cnt;
int summu[MAXN], ans;

void sieve(int m) {
    isNotPrime[0] = isNotPrime[1] = true;
    mu[1] = 1;
    for (int i = 2; i <= m; ++i) {
        if (!isNotPrime[i]) {
            primes[++cnt] = i;
            mu[i] = -1;
        }
        for (int j = 1; j <= cnt && i * primes[j] <= m; ++j) {
            isNotPrime[i * primes[j]] = true;
            if (i % primes[j] == 0) {
                mu[i * primes[j]] = 0;
                break;
            }
            mu[i * primes[j]] = -mu[i];
        }
    }
}

int query(int n, int m) {
    int res = 0;
    for (int l = 1, r; l <= std::min(n, m); l = r + 1) {
        r = std::min(n / (n / l), m / (m / l));
        res += (summu[r] - summu[l - 1]) * (n / l) * (m / l);
    }
    return res;
}

int main() {
    sieve(50000);
    std::partial_sum(mu + 1, mu + 50000 + 1, summu + 1);
    scanf("%d", &N);
    while (N--) {
        scanf("%d %d %d %d %d", &a, &b, &c, &d, &k);
        ans = query(b / k, d / k) - query(b / k, (c - 1) / k) - query((a - 1) / k, d / k) + query((a - 1) / k, (c - 1) / k);
        printf("%d\n", ans);
    }
    return 0;
}

  2. YY 的 GCD

    题意：求

    ∑i=1n∑j=1m[gcd⁡(i,j) is prime] \sum_{i = 1}^n \sum_{j = 1}^m [\gcd(i, j) \text{ is prime}] i=1∑n​j=1∑m​[gcd(i,j) is prime]

    思路：

    设 P\mathbb{P}P 为质数集。不妨令 n≤mn \le mn≤m。

    ∑i=1n∑j=1m[gcd⁡(i,j)∈P]=∑p∈P∑i=1n∑j=1m[gcd⁡(i,j)=p]=∑p∈P∑d=1⌊n/p⌋μ(d)⌊npd⌋⌊mpd⌋ \sum_{i = 1}^n \sum_{j = 1}^m [\gcd(i, j) \in \mathbb{P}] = \sum_{p \in \mathbb{P}}\sum_{i = 1}^n \sum_{j = 1}^m [\gcd(i, j) = p] \\ = \sum_{p \in \mathbb{P}} \sum_{d = 1}^{\lfloor n / p \rfloor} \mu(d) \left \lfloor \frac{n}{pd} \right \rfloor \left \lfloor \frac{m}{pd} \right \rfloor i=1∑n​j=1∑m​[gcd(i,j)∈P]=p∈P∑​i=1∑n​j=1∑m​[gcd(i,j)=p]=p∈P∑​d=1∑⌊n/p⌋​μ(d)⌊pdn​⌋⌊pdm​⌋

    直接枚举显然复杂度不对。设 t=pdt = pdt=pd，则

    ∑p∈P∑d=1⌊n/p⌋μ(d)⌊npd⌋⌊mpd⌋=∑t=1n∑p∣t,p∈Pμ(tp)⌊nt⌋⌊mt⌋=∑t=1n⌊nt⌋⌊mt⌋∑p∣t,p∈Pμ(tp) \sum_{p \in \mathbb{P}} \sum_{d = 1}^{\lfloor n / p \rfloor} \mu(d) \left \lfloor \frac{n}{pd} \right \rfloor \left \lfloor \frac{m}{pd} \right \rfloor = \sum_{t = 1}^n \sum_{p \mid t, p \in \mathbb{P}}\mu(\frac{t}{p}) \left \lfloor \frac{n}{t} \right \rfloor \left \lfloor \frac{m}{t} \right \rfloor \\ = \sum_{t = 1}^n \left \lfloor \frac{n}{t} \right \rfloor \left \lfloor \frac{m}{t} \right \rfloor \sum_{p \mid t, p \in \mathbb{P}}\mu(\frac{t}{p}) p∈P∑​d=1∑⌊n/p⌋​μ(d)⌊pdn​⌋⌊pdm​⌋=t=1∑n​p∣t,p∈P∑​μ(pt​)⌊tn​⌋⌊tm​⌋=t=1∑n​⌊tn​⌋⌊tm​⌋p∣t,p∈P∑​μ(pt​)

    其中 ⌊nt⌋⌊mt⌋\left \lfloor \frac{n}{t} \right \rfloor \left \lfloor \frac{m}{t} \right \rfloor⌊tn​⌋⌊tm​⌋ 可以整除分块，设 f(t)=∑p∣t,p∈Pμ(tp)f(t) = \sum_{p \mid t, p \in \mathbb{P}}\mu(\frac{t}{p})f(t)=∑p∣t,p∈P​μ(pt​) ，这个东西可以对于每个质数枚举其倍数 ttt，给 f(t)f(t)f(t) 的值累加上 μ(tk)\mu(\frac{t}{k})μ(kt​) 就可以统计出 f(t)f(t)f(t) 的值，再求个前缀和就能和整除分块一起计算了。复杂度 O(能过)O(能过)O(能过)，具体我不会算。

    #include <algorithm>
#include <cstdio>
#include <numeric>

const int MAXN = 10000000 + 5;

int T, n, m;
int primes[700000], mu[MAXN], g[MAXN], cnt;
long long sumg[MAXN];
bool isNotPrime[MAXN];

void sieve(int s) {
    isNotPrime[1] = true;
    mu[1] = 1;
    for (int i = 2; i <= s; ++i) {
        if (!isNotPrime[i]) {
            primes[++cnt] = i;
            mu[i] = -1;
        }
        for (int j = 1; j <= cnt && i * primes[j] <= s; ++j) {
            isNotPrime[i * primes[j]] = true;
            if (i % primes[j] == 0) {
                mu[i * primes[j]] = 0;
                break;
            }
            mu[i * primes[j]] = -mu[i];
        }
    }
}

void calcg(int s) {
    for (int i = 1; i <= cnt; ++i) {
        for (int j = 1; primes[i] * j <= s; ++j) {
            g[primes[i] * j] += mu[j];
        }
    }
}

long long solve(int n, int m) {
    long long res = 0;
    for (int l = 1, r, s = std::min(n, m); l <= s; l = r + 1) {
        r = std::min(n / (n / l), m / (m / l));
        res += 1ll * (n / l) * (m / l) * (sumg[r] - sumg[l - 1]);
    }
    return res;
}

int main() {
    sieve(MAXN - 5);
    calcg(MAXN - 5);
    std::partial_sum(g + 1, g + MAXN - 4, sumg + 1);
    scanf("%d", &T);
    while (T--) {
        scanf("%d %d", &n, &m);
        printf("%lld\n", solve(n, m));
    }
    return 0;
}
